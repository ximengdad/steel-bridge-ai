# 柳林桥图纸解析说明

## 当前能力

### ✅ 已实现
API目前可以提供：
- **图纸元数据**：图号、图名、分类、文件路径
- **图纸检索**：按关键词/分类搜索
- **施工草图**：基于预设参数生成SVG流程图

### ❌ 限制
API**不能**直接读取DWG文件内部的：
- 几何尺寸
- 坐标点
- 图层详细信息
- 尺寸标注数值

---

## 为什么DWG文件难以直接解析？

DWG是AutoCAD的私有二进制格式，需要：
1. **专用解析库** - 如 Teigha、AutoCAD SDK（商业/复杂）
2. **格式转换** - DWG → DXF（开放格式）→ 解析

---

## 解决方案（推荐）

### 方案1：DWG转DXF + ezdxf解析（免费）

#### 步骤1：安装依赖
```bash
pip install ezdxf
```

#### 步骤2：转换DWG为DXF
下载 **ODA File Converter**（免费）：
- 官网：https://www.opendesign.com/guestfiles/oda_file_converter
- 支持批量转换

```bash
# 命令行转换示例
odafileconverter "." "." "ACAD2018" "DXF" "0" "1"
```

#### 步骤3：API自动解析
转换后的DXF文件可被API自动解析：
```bash
GET /drawings/S2-1-2-5/parse
```

返回：
```json
{
  "drawingNo": "S2-1-2-5",
  "parseStatus": "success",
  "content": {
    "layers": ["轮廓线", "尺寸线", "中心线"],
    "entities": {
      "lines": 150,
      "circles": 20,      // 可能是螺栓孔
      "dimensions": 45    // 尺寸标注
    },
    "dimensions": [
      {"type": "circle", "radius": 12.5, "center": [100, 200]},
      {"type": "dimension", "text": "1200", "layer": "尺寸线"}
    ],
    "boundingBox": {
      "width": 8500,
      "height": 3200
    }
  }
}
```

### 方案2：在线DWG查看器集成
如果不想转换文件，可以：
1. 使用 **AutoCAD Web** 或 **ViewerJS** 嵌入查看器
2. 前端直接渲染DWG文件
3. API只提供文件路径，不解析内容

---

### 方案3：手动提取关键参数
对于最重要的图纸（如Z1-Z10拱肋节段），可以：
1. 人工读取DWG中的关键尺寸
2. 录入到知识图谱的 `sb:parameters` 字段
3. API直接返回预存参数（无需解析）

已在 `kg_liulin_bridge.jsonld` 预留参数字段：
```json
{
  "@id": "sb:MainArch_Z1",
  "@type": "sb:Component",
  "sb:componentId": "Z1",
  "sb:parameters": {
    "length": 8500,
    "width": 1200,
    "height": 1800,
    "plateThickness": 20
  }
}
```

## 常见问题

**Q: 为什么不用Python直接读DWG？**
A: DWG是闭源格式，Python没有成熟的免费DWG解析库。必须转换或商业授权。

**Q: 转换DXF会丢失信息吗？**
A: 大部分几何信息保留，但某些高级特性可能丢失。对于施工图纸（几何+尺寸）完全够用。

**Q: 有多少图纸需要转换？**
A: 柳林桥共72张DWG，批量转换约需5-10分钟。

**Q: 可以只转换关键图纸吗？**
A: 可以。建议优先转换：
- 拱肋节段构造图（Z1-Z10, C1-C5）
- 主梁节段构造图
- 关键连接节点图
