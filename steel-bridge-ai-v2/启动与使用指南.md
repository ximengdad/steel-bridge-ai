# 柳林桥钢箱梁智能助手 - 启动与使用指南

## 快速启动（3步）

### 第1步：启动后端API服务

```bash
cd /root/.openclaw/workspace/一航局课题交付包/steel-bridge-ai-v2/01_交付物1_知识图谱服务
python3 main.py
```

你会看到：
```
INFO:     Started server process [xxxxx]
INFO:     Uvicorn running on http://0.0.0.0:8000
✅ 柳林桥知识图谱加载成功: 62 个实体
  - 图纸: 46 张
  - 构件系统: 6 个
  - 工序: 5 个
  - 规范: 4 个
```

**保持这个窗口运行**，这是API服务。

---

### 第2步：打开Web前端

在浏览器中打开文件：
```
file:///root/.openclaw/workspace/一航局课题交付包/steel-bridge-ai-v2/02_交付物2_Web端大模型/frontend/index.html
```

或者直接在文件管理器中双击 `index.html`

---

### 第3步：使用系统

现在你可以：

#### 1. 查看图纸列表
- 左侧菜单点击 "📐 图纸检索"
- 默认显示全部分类的图纸
- 点击分类标签筛选：下部结构、拱肋结构、主梁系统...

#### 2. 查看图纸尺寸
- 找到任意图纸卡片（如 S2-1-2-5）
- 点击 **"📐 查看尺寸"** 按钮
- 展开显示详细尺寸：长度、宽度、板厚、圆孔数量等

#### 3. 搜索图纸
- 在搜索框输入关键词，如：
  - `Z1` - 查找Z1节段图纸
  - `拱肋` - 查找拱肋相关图纸
  - `主梁` - 查找主梁相关图纸

#### 4. 施工指导
- 左侧菜单点击 "🏗️ 施工指导"
- 选择构件系统：拱肋安装/主梁制作/吊索安装...
- 查看相关图纸和施工工序

#### 5. 生成施工草图
- 左侧菜单点击 "✏️ 草图生成"
- 选择草图类型：拱肋安装/主梁制作/桥面系施工
- 自动生成SVG流程图

#### 6. AI智能问答
- 左侧菜单点击 "💬 智能问答"
- 输入问题，如：
  - "Z1节段尺寸多少？"
  - "拱肋安装需要什么图纸？"
  - "柳林桥有多少张图纸？"

## 常见问题

**Q: 浏览器提示"无法访问此网站"？**
A: 检查后端API服务是否已启动（第一步的终端窗口要保持运行）

**Q: 图纸尺寸显示"需要转换格式"？**
A: 部分图纸只有DWG没有DXF，已生成DXF的59张图纸可正常显示尺寸

**Q: 如何停止服务？**
A: 在运行API服务的终端窗口按 `Ctrl+C`
